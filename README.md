# The Date

A cinematic, interactive date request — carefully designed after an unreasonable amount of overthinking.

## Sections

- **Hero** — The opening question with ambient particles and scroll indicator
- **Doodles** — Terms & conditions cards with hand-drawn SVG doodles and pastel colors
- **Confession** — Image card carousel with personal photos and heartfelt text
- **Memories** — Floating organic bubble universe with photos and future memory placeholders
- **Decision** — The final yes/no with a puppy doodle, pixelated hearts, and growing buttons
- **Success** — Dress code card with hand-drawn SVG and closing message

## Features

- Snap scroll between sections
- Fixed parallax background with crossfading cosmic indigo palette
- Loading screen with staggered messages and skip button
- Bottom navigation bar with scroll-aware highlighting
- Horizontal card carousels with rotation animations
- Responsive design for mobile

## Tech Stack

- TanStack Start + React 19
- Vite 7
- Tailwind CSS 4
- Framer Motion
- Cloudflare Workers (deployment)

## Development

```bash
bun install
bun run dev
```

Requires Node.js 20.19+ or 22.12+.
